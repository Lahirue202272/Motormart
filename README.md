## Mototrmart is a production-ready Django web application built for a used car dealership.
## It allows users to browse cars, filter listings, view details, send inquiries, and manage accounts with secure authentication.

## The project includes:

PostgreSQL database configuration

Custom Django Admin (logos, styles, CKEditor, Multiselect fields)

Search & filtering system (model, year, price, keywords)

User login, registration, logout

Social authentication (Google & Facebook)

Inquiry system with email notifications

Pagination, rich-text support, and image galleries

Deployment-ready setup with Heroku, Gunicorn & Whitenoise

Database import/export (dumpdata & loaddata)

Static & media files support
